# Exoplayer in recyclerview and storyview [![](https://jitpack.io/v/hk1089/ExoPlayer-Action.svg)](https://jitpack.io/#hk1089/ExoPlayer-Action) ![](https://img.shields.io/github/languages/top/hk1089/ExoPlayer-Action) 

A sutable solution for exoplayer library conflict. In this library you can auto play video in recyclerview also if you want to implement story like instagram or whatsapp you can use it.

# Usage


**Dependencies**
- **Step 1: Add the JitPack repository in your project build.gradle file**
```bash
allprojects {
	    repositories {
		    ...
		    maven { url 'https://jitpack.io' }
	    }
    }
```

 **Step 2: Add the dependency in your app module build.gradle file**
```bash

implementation 'com.github.hk1089:ExoPlayer-Action:x.x.x'
	
```
